Data Extraction & Cleaning (SQL)

SQL Query: Joining Tables 
-- Gộp bảng Customers và Geography
SELECT CustomerID, CustomerName, Email, Gender, Coutry, City
FROM customers AS cu
LEFT JOIN geography AS ge ON ge.GeographyID = cu.GeographyID;
SQL Query: Product Normalization 
SELECT ProductID, ProductName, Price,
  CASE
    WHEN Price < 50 THEN 'Low_price'
    WHEN Price BETWEEN 50 AND 100 THEN 'Medium_price'
    ELSE 'High_price'
  END AS Categogy_price
FROM products;

Python Script: VADER Sentiment
def classify_sentiment(score, rating): 
    if score > 0.05:
        return 'tích cực' if rating >= 4 else 'khá tích cực'
    elif score < -0.05:
        return 'tiêu cực'
    else:
        return 'trung lập'
