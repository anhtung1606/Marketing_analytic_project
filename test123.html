Phần 1: Ý tưởng & Kế hoạch

Trong dự án này, tôi đặt mục tiêu giải quyết bài toán sụt giảm doanh thu tại các chi nhánh bán lẻ. Cụ thể, những việc tôi sẽ làm bao gồm:

  • Xác định các giao dịch có giá trị bất thường (Outliers).
  • Tính toán tỷ lệ tăng trưởng doanh thu theo từng tháng (MoM Growth).
  • Phân tích hành vi mua sắm của các nhóm khách hàng khác nhau.
  • Đề xuất danh mục sản phẩm cần khuyến mãi dựa trên dữ liệu tồn kho.
01 Gộp bảng Customers và Geography 
-- Gộp theo ID geography để tạo ra bảng mới chứa đầy đủ thông tin vị trí
SELECT 
    cu.CustomerID, cu.CustomerName, cu.Email, cu.Gender, 
    ge.Country, ge.City 
FROM customers AS cu
LEFT JOIN geography AS ge ON ge.GeographyID = cu.GeographyID;
02 Phân loại giá Sản phẩm (Bucket Analysis) 
SELECT ProductID, ProductName, Price,
CASE
    WHEN price < 50 THEN 'Low_price'
    WHEN price BETWEEN 50 AND 100 THEN 'Medium_price'
    ELSE 'High_price'
END AS Category_price
FROM products;
03 Làm sạch và Chuẩn hóa Engagement Data 
SELECT
    EngagementID, ContentID, CampaignID,  
    UPPER(REPLACE(ContentType, 'Socialmedia', 'Social Media')) AS ContentType,  
    LEFT(ViewsClicksCombined, CHARINDEX('-', ViewsClicksCombined) - 1) AS Views,  
    RIGHT(ViewsClicksCombined, LEN(ViewsClicksCombined) - CHARINDEX('-', ViewsClicksCombined)) AS Clicks 
FROM dbo.engagement_data  
WHERE ContentType != 'Newsletter';

# Hàm phân loại dựa trên score và rating của người dùng
def classify_sentiment(score, rating): 
    if score > 0.05:
        return 'Tích cực' if rating >= 4 else 'Khá tích cực'
    elif score < -0.05:
        return 'Tiêu cực' if rating <= 2 else 'Khá tiêu cực'
    else:
        return 'Trung lập'
